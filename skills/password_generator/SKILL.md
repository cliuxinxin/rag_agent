## 功能描述

这个技能可以生成强密码，包含以下字符类型：
- 大写字母 (A-Z)
- 小写字母 (a-z)
- 数字 (0-9)
- 符号 (!@#$%^&*()_+-=[]{}|;:,.<>?)

## 使用方法

### 基本用法
```bash
# 生成12位密码
password_generator 12
```

### 参数说明
- 第一个参数：密码长度（必需）
  - 最小长度：8
  - 最大长度：50
  - 推荐长度：12-16

## 示例

```bash
# 生成16位密码
password_generator 16
# 输出：A3b$kL9@mN2#pQ8!
```

## 安全建议

1. 使用至少12位长度的密码
2. 定期更换密码
3. 不要在不同网站使用相同密码
4. 考虑使用密码管理器

## 技术细节

密码生成算法确保：
1. 每种字符类型至少出现一次
2. 字符随机分布
3. 使用安全的随机数生成器
